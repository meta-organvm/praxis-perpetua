# AMP Lab Media -- Content Calendar

## Overview

This is the rolling content calendar for AMP Lab Media's V2 relaunch, focused exclusively on the Object Lessons series. The strategy is built on three pillars: consistent long-form episodes, aggressive Shorts extraction, and community engagement once subscriber thresholds unlock those features.

**Target cadence:** 2 episodes/month (biweekly)
**Shorts cadence:** 2/week (from new episodes + back catalog)
**Community posts:** 3/week (once 500 subs unlocks the Community tab)

### Content Type Definitions

**Long-form (Episode):** 10-12 minute Object Lesson following `templates/object-lesson-episode.md`

**Shorts:** 15-60 second vertical clips. Three types:
1. **Episode clips** -- strongest visual moments from new episodes, text overlay linking to full video
2. **Back catalog clips** -- clips from existing episodes, re-edited for vertical format
3. **Standalone micro-essays** -- rare; only when a single clip + 1-2 sentence insight works on its own

**Community posts:** Three types:
1. **Polls** -- "Which object should we cover next?" or "Which film used [object] best?"
2. **Behind the scenes** -- research process, edit timeline, clip sourcing stories
3. **Engagement** -- "What did we miss?", fan submissions, film recommendations, challenge questions
