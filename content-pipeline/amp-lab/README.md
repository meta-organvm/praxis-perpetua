# AMP Lab Media -- Content Pipeline

## What This Is

AMP Lab Media (@AmpLabMedia) is a YouTube channel co-created with Chris that produces video essays examining recurring objects across film history through compiled clips. The flagship series, **Object Lessons**, traces a single object (milk, mirrors, cigarettes) across decades of cinema to reveal how directors encode meaning through physical things.

## Human Gates

The following decisions and actions require human involvement -- they cannot be automated or delegated to AI agents.

### Requires Chris (co-creator)

- [ ] **Creative direction approval** -- agreement on V2 format, tone, and scope
- [ ] **Object selection sign-off** -- final pick for each episode's subject
- [ ] **Narration recording** -- voice performance for each episode
- [ ] **Edit review** -- final cut approval before upload
- [ ] **Channel branding decisions** -- name, logo, banner refresh for V2

### Requires Account Access

- [ ] **YouTube Studio access** -- upload, SEO settings, analytics, monetization
- [ ] **Google AdSense setup** -- required for YPP monetization once eligible
- [ ] **Channel verification** -- custom URL, community tab access
- [ ] **Content ID dispute resolution** -- responding to copyright claims on fair-use clips

### Requires External Coordination

- [ ] **Music licensing/commission** -- ETCETER4 original scores for each episode
- [ ] **Collaboration outreach** -- reaching out to other video essayists for cross-promotion
- [ ] **Patreon setup** -- creating tiers, rewards, and payment infrastructure
