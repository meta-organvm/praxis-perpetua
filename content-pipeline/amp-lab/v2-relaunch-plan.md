## Phase 1: Relaunch (Months 1-3)

### Content

- [ ] **Comeback video** (Week 1): 2-3 min statement of intent. "We're back -- here's what Object Lessons is now." Not a behind-the-scenes vlog -- a cinematic declaration. Show the format evolution: pure compilation becomes narrated hybrid. End with a preview montage of upcoming objects.

- [ ] **Episode 1** (Week 2): First new Object Lesson in the narrated hybrid format. Must be the STRONGEST possible object.
  - **Recommendation: Milk** -- iconic (A Clockwork Orange, Get Out, Inglourious Basterds, No Country for Old Men), existing search volume, culturally loaded (innocence, corruption, domesticity). Now You See It did a video but not a definitive compilation. Object Lessons can own this.

- [ ] **Episode 2** (Week 4): **Recommendation: Mirrors** -- universal symbol across every genre and era, enormous cross-director range (Hitchcock, Kubrick, Aronofsky, Park Chan-wook), visually stunning material, inherent psychological depth (reflection, identity, vanity, truth).

- [ ] **Episode 3** (Week 6): **Recommendation: Cigarettes** -- declining frequency across decades tells a cultural story (from ubiquitous glamour to coded rebellion to near-disappearance). Political dimension adds urgency. The object's absence becomes as meaningful as its presence.

- [ ] **Episode 4** (Week 8): **Clocks/Watches** -- time, mortality, urgency, countdown. Hitchcock to Nolan. The ticking clock is cinema's most universal tension device.

- [ ] **Episode 5** (Week 10): **Doors** -- threshold symbolism, power dynamics, the boundary between worlds. Opening, closing, slamming, locking. The Godfather's final shot is one of the most analyzed in cinema history.

- [ ] **Episode 6** (Week 12): **Guns** -- the most loaded object in American cinema. Political, cultural, and aesthetic dimensions. Every genre, every era, every continent.

### Shorts Pipeline

- 2 Shorts/week from new episodes + back catalog
- Each Short: pinned comment linking to full video, text overlay "Full video on my channel"
- Formats:
  - "Guess the Film" (show object, reveal movie)
  - Side-by-side comparisons across decades
  - Most surprising appearance of an object
  - "One object, five directors" -- same object in Kubrick vs. Spielberg vs. Lynch vs. Villeneuve vs. Bong

## ORGANVM Integration

This plan maps across four organs:

| Component | Organ | Repo (existing or to scaffold) |
|-----------|-------|-------------------------------|
| Process governance (this plan, SOPs, reviews) | META | `praxis-perpetua` |
| Creative production (scripts, edits, research) | ORGAN-II (Poiesis) | `amp-lab-media` (to scaffold) |
| Distribution automation (SEO, Shorts, scheduling) | ORGAN-VII (Kerygma) | `amp-lab-kerygma-profile` (to scaffold) |
| Film-scrubbing tooling | ORGAN-III (Ergon) | `object-lessons-scrubber` (to scaffold) |

### Tooling Roadmap (ORGAN-III)

The automated film-scrubbing system is the long-term competitive moat:

| Phase | Scope | Timeline |
|-------|-------|----------|
| Phase 1 | Manual clip extraction with structured database (spreadsheet/YAML per object, timestamped entries, source tracking) | Months 1-3 |
| Phase 2 | Semi-automated detection using existing computer vision models (YOLO, CLIP) fine-tuned on cinema stills | Months 4-8 |
| Phase 3 | Full automation: given a film file and target object, extract timestamped clips with context, tag, and add to searchable archive | Year 2+ |

Each phase increases production efficiency and becomes a defensible asset. By Phase 3, no other creator can assemble clips at this scale with this efficiency.
