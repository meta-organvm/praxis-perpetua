# Lens Catalog

Reference table for the 21 organizational lenses identified in the universal hierarchy synthesis. These lenses are **summoned per-task, not held permanently** — see Y12 in [derived-principles.md](derived-principles.md) and [SOP--dynamic-lens-assembly.md](../standards/SOP--dynamic-lens-assembly.md) for the assembly protocol.

**Source:** [Universal Hierarchy Synthesis](../research/2026-03-08-universal-hierarchy-synthesis.md), §III

---

| Lens | Summon When | Primary Insight | Primary Critique |
|------|------------|-----------------|------------------|
| **Mathematics** | Verifying state machine completeness, formal properties | DFA/category theory reveals structural gaps | Cannot capture meaning, only structure |
| **Information Theory** | Measuring complexity, diagnosing bottlenecks | Entropy and compression ratios | Cannot capture value, only surprise |
| **Metaphysics** | Ontological confusion (is this a tool or organism?) | Permission for ontological polymorphism | Can paralyze with infinite alternatives |
| **Thermodynamics** | Diagnosing decay, energy budgets | Entropy accumulation is measurable | Treats all disorder as equivalent |
| **Quantum Mechanics** | Understanding measurement effects | State is genuinely indefinite between observations | Analogies can overextend to absurdity |
| **Chemistry** | Planning repo composition, auditing bonds | Valence rules constrain valid combinations | Not all organizational bonds are chemical |
| **Cosmology** | Understanding temporal arc | The system has epochs, not just states | Cosmological narratives can mythologize mundane history |
| **Chaos** | Harvesting novelty, protecting creativity | The boundary between order and disorder IS the creative engine | Can justify any disorder as "productive" |
| **Neuroscience** | Cognitive load, attention allocation, rest | 7±2 is a hard limit; forgetting is a feature | Reductive about meaning and purpose |
| **Ecology** | Capacity planning, archival decisions, sprawl | Carrying capacity is finite; decomposition is necessary | Ecological metaphors can naturalize what is designed |
| **Organismal Biology** | Health diagnostics, system architecture | Homeostasis demands automatic response, not just monitoring | The organism metaphor can anthropomorphize software |
| **Cellular Biology** | Template design, context sync, repo creation | DNA/expression/apoptosis map precisely to seed.yaml/CLAUDE.md/ARCHIVED | Cells don't have agency; repos are artifacts of choice |
| **Academia** | Promotion decisions, quality review, onboarding | Tenure track maps to promotion pipeline | Academic timescales are too slow for a startup |
| **Belief Systems** | Doctrine maintenance, principle consistency | Sacred texts need amendment processes | Sacralization can prevent needed changes |
| **Cultural Expression** | Aesthetic decisions, naming, identity | House style requires space for variation | Culture can excuse poor engineering |
| **Infrastructure** | Reliability, redundancy, failure analysis | Critical path demands explicit triage ordering | Infrastructure thinking deprioritizes innovation |
| **Governance** | Authority questions, rule creation, enforcement | Constitutional/statutory/case law hierarchy | Governance expands to fill available attention |
| **Economics** | Resource allocation, opportunity cost, trade-offs | Attention is scarce; allocation has opportunity costs | Economic logic can commodify what shouldn't be commodified |
| **The Technium** | Technology adoption, tool evaluation | Infrastructure has autonomous growth tendencies | Technium analysis can excuse feature creep |
| **The Noosphere** | External impact, contribution, publication | Internal convergence without external transmission is solipsism | Noospheric ambition can justify anything as "contributing to collective intelligence" |
| **Sociology** | Self-critique, signal vs. function analysis | Identifies performative vs. functional governance | Sociology deconstructs but does not construct |
