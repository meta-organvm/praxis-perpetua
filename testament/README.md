# The Mirror Protocol — Network Testament

> context[current-work] > relevant[open-source] > expand[network]

This directory is the accumulating evidence of a principle: that genuine work,
mapped to its reflections in the open-source world, produces genuine relationships.
Not networking. Not self-promotion. Mirrors.

Everything here was produced by working — then looking up from the work to see who
else is in the room.

## The Three Lenses

Every ORGANVM repo reflects something external through three equal lenses:

- **Technical** — projects whose code runs inside ORGANVM
- **Parallel** — projects solving similar problems differently
- **Kinship** — communities aligned in values, not technology

## The Four Forms

Engagement takes four simultaneous forms, not a progression:

- **Presence** — be in the room
- **Contribution** — add value
- **Dialogue** — be in conversation
- **Invitation** — become a destination

## The Principle

We are not one thing. We do not take the shape of one thing. Everything is a mirror.

A pull request upstream and joining a conversation about creative autonomy carry
equal weight. A bug report on a linting tool and attending a digital humanities
gathering are both acts of showing up. The network emerges from the full surface
of the work — all 8 organs, all 113 repos, all the way down.

## Raw Evidence

The ledger lives at `~/.organvm/network/ledger.jsonl` — append-only, never purged.
The `network-map.yaml` files live alongside `seed.yaml` in each repo.
This directory holds the synthesis — what the ledger and maps, read together, reveal.
