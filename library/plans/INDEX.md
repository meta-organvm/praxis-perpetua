# Plan Library — Index

Central catalog of all plans across the ORGANVM workspace. Plans stay project-local (`.claude/plans/` in each repo). This index provides discovery.

**Status:** Awaiting `organvm plans catalog --write` to generate `index.json` and `by-domain/` pointer files.

## Discovery

```
organvm plans catalog --write     # Build/rebuild the catalog
organvm plans search <query>      # Search by title, tags, domain, organ
```
